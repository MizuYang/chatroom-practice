<h2>聊天室</h2>

- 模擬 Line 聊天室介面
- 自己的對話氣泡為綠色、對方則為白色 (Line 配色)
- 可使用 Emoji 功能
- 按下送出可發送訊息
- 有新對話時，對話滾動條自動置底
